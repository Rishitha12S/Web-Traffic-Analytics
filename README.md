# Web Traffic Analytics

## Objective
Analyze website traffic data to understand user behavior and engagement patterns.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Features
- Page view analysis
- Session duration analysis
- Bounce rate analysis
- Traffic source evaluation
- Data visualization
- User engagement insights

## Project Structure

  Web-Traffic-Analytics/
│
├── data/
│   └── website_traffic.csv
│
├── outputs/
│
├── src/
│   └── analysis.py
│
├── requirements.txt
├── README.md


## How to Run

Install required libraries:

```bash
pip install -r requirements.txt
````

Run the project:

```bash
python src/analysis.py
```

## Output

The project generates:

* Traffic source charts
* Bounce rate visualization
* Session duration graphs
* Exit page analysis
* User journey insights
